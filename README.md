
# 📊 Tableau Data Visualisation Project – Employment, Retail & Health Insights

## 📘 Overview
This Tableau project was completed during a **Data Technician Bootcamp**, focusing on **data analysis, business intelligence (BI), and interactive dashboard development**.

The project integrates multiple real-world datasets:
- 📌 EMSI Job Change (UK)
- 📌 GB Employment Dataset
- 📌 Health (Global Gapminder Dashboard)
- 📌 Spotify Dataset (trend and behavioural analysis)

The objective was to analyse **employment trends, retail/sales-style patterns, and health data**, and transform raw datasets into **interactive dashboards and compelling data stories**.

## 📈 Data Visualisation Techniques

A variety of chart types were used to present insights effectively:

- 📊 **Bar Charts** – Comparing job changes, employment rates, and category performance  
- 📉 **Line Charts** – Analysing trends over time (employment, health indicators, music trends)  
- 🥧 **Pie Charts** – Showing proportional splits (e.g., workforce distribution, categories)  
- 🗺️ **Map Visualisations** – Regional analysis across the UK  
- 🔵 **Scatter Plots** – Correlation analysis (e.g., BMI vs Life Expectancy, popularity trends)

---

## 🔍 Project Analysis

### 📌 Employment & Workforce Insights
- Analysed **job growth and decline trends across UK regions**
- Identified **high-demand industries and workforce shifts**
- Compared **regional employment performance using GB Employment data**

---

### 📌 Retail / Sales-Style Insights
- Applied sales-style analysis techniques (aggregation, segmentation, KPIs)
- Identified **patterns in performance and category behaviour**
- Used calculated metrics to simulate **business reporting scenarios**

---

### 📌 Health Dashboard Insights
- Visualised **life expectancy trends across countries**
- Analysed relationships between **population, BMI, and life expectancy**
- Highlighted global health disparities for decision-making use cases 

---

### 📌 Spotify Dataset Insights
- Explored **music trends and user behaviour patterns**
- Analysed correlations between **popularity, energy, and other features**
- Used visualisations to identify **top-performing genres and characteristics**
